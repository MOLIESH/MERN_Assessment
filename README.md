# Product Management App

## Tech Stack
- React
- Node.js
- Express
- MySQL

## Setup Instructions

### Backend
1. cd backend
2. npm install
3. Create MySQL database `products_db`
4. Run migrations/products.sql
5. node app.js

### Frontend
1. cd frontend
2. npm install
3. npm run dev

## Features
- Reusable sortable & filterable table
- REST API with soft delete
